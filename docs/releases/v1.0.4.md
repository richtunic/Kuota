## Kuota v1.0.4

### Cambios principales

- Login nativo de Codex en macOS y Windows: Kuota ejecuta `codex-auth login --device-auth` en segundo plano y ya no requiere Terminal, CMD ni PowerShell visibles para agregar cuentas.
- El codigo de autorizacion se copia automaticamente al portapapeles antes de abrir OpenAI. En macOS usa `pbcopy`; en Windows usa el portapapeles del sistema con PowerShell oculto.
- Kuota abre automaticamente la pagina de autorizacion de OpenAI en el navegador predeterminado despues de detectar y copiar el codigo.
- El modal de autorizacion ahora permite cancelar el flujo y cierra el proceso hijo para no dejar logins pendientes.
- La deteccion del codigo fue reforzada para evitar falsos positivos como `ChatGPT` y para aceptar formatos futuros con guiones, underscores, letras o numeros.
- Si OpenAI limita temporalmente la generacion de codigos, Kuota muestra un mensaje amigable en lugar de fallar con un error tecnico.
- En Windows, Kuota verifica e instala automaticamente las dependencias necesarias para el flujo de login: `@loongphy/codex-auth` y `@openai/codex`.

### Documentacion y reglas

- README actualizado con el comportamiento del login nativo en macOS y Windows.
- Queda documentado que las releases oficiales compilan solo macOS y Windows hasta nuevo aviso.
- Cada version nueva debe incluir cambios visibles para el usuario en GitHub Releases, ademas de commits descriptivos.
